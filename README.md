## Introduction

此为非官方整理用于使用 CocoaPods 部署百度推送SDK的项目

## Version

1.4.6

## Requirements

iOS 5.1

## Installation

BPushSDK is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
# Equivalent to normal version
pod 'BPushSDK'

# Normal version
pod 'BPushSDK/normal'
# IDFA version
pod 'BPushSDK/idfa'
```
